<h2>🚀 Features</h2>
<ul>
<li><strong>Interactive Cloud Canvas</strong>: The background isn't static! Move your mouse or drag your finger across the screen to paint fluffy, dreamy clouds over a soft pink sky.</li>
<li><strong>Runaway "No" Button</strong>: Trying to say no? Good luck! The red button actively dodges clicks and touches, making it nearly impossible to reject the proposal.</li>
<li><strong>Dynamic Layout</strong>: Every time you try to click "No", the "Yes" button grows bigger and bigger, ensuring there are no overlapping elements and the layout stays perfectly clean.</li>
<li><strong>Customizable via URL</strong>: Make it personal in seconds! Just add <code>?name=YourName</code> to the URL, and the webpage will automatically update its title just for them.</li>
<li><strong>Secret "Storm" Easter Egg</strong>: What happens if someone is stubborn enough to actually catch the running button? A hidden storm mode triggers to show just how heartbroken you'd be!</li>
<li><strong>100% Client-Side</strong>: Built with pure HTML, CSS, and Vanilla JavaScript. No servers, no tracking, just love and code.</li>
</ul>

<h2>🛠️ How it works</h2>
<ol>
<li><strong>Load the page:</strong> The URL parameters are parsed to inject the requested name directly into the DOM.</li>
<li><strong>Dynamic Background:</strong> An invisible HTML5 <code>&lt;canvas&gt;</code> element captures <code>mousemove</code> and <code>touchmove</code> events. It draws translucent radial gradients to simulate clouds that slowly blend into the pink background.</li>
<li><strong>Interaction Logic:</strong> An array of safe, verified Giphy links is used to cycle through adorable crying animations when the user tries to decline.</li>
<li><strong>The Escape Mechanics:</strong> After 6 attempts, the "No" button switches to <code>position: fixed</code>. JavaScript calculates safe screen boundaries and instantly teleports the button to a random <code>X/Y</code> coordinate whenever a <code>mouseover</code> or <code>touchstart</code> event is detected.</li>
</ol>
